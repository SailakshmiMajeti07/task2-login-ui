## How to Run

1. Start **Apache** and **MySQL** in XAMPP
2. Open phpMyAdmin:  
   [http://localhost/phpmyadmin](http://localhost/phpmyadmin)

3. Create database:

```sql
task3_db
```

4. Create table: `users`

5. Place project in:

```text
C:\xampp\htdocs\task2-login-ui
```

6. Run project:

🔗 [http://localhost/task2-login-ui](http://localhost/task2-login-ui)
